我的英语水平大约是「词汇量约 6000 个」。
对「词汇量约 6000 个」来说**可能不熟**的词，我会先在句子里用中括号标注。

当我给你输入时，请只做“翻译已标注词”这一件事。

=============== 输入说明 ===============

1. 预处理后的句子（以这个为准）：
{{ANNOTATED_TEXT}}

2. 原始句子（仅供语境参考，不用于改写）：
{{TEXT}}

3. 待翻译词列表（全部小写，去重）：
{{TARGET_WORDS}}

=============== 任务要求 ===============

1. 你只能翻译 `{{TARGET_WORDS}}` 里出现的词。
2. 你不能新增要翻译的词，也不能漏掉词。
3. `sentence` 字段必须与输入的 `{{ANNOTATED_TEXT}}` 完全一致（逐字符一致）。
4. `meaning` 的键必须与 `{{TARGET_WORDS}}` 完全一致，键名保持小写。
5. 如果某个目标词是人名（人名、姓氏、昵称），不要翻译其词义，直接写“人名，不翻译”。
6. 对非人名词，每条释义都要结合当前句子语境，用自然中文解释该词在句中的含义。
7. 不要输出语法分析、备注、额外说明。

=============== 输出格式（严格遵守） ===============

最终只输出一个合法 JSON 对象：

{
  "sentence": "这里必须与 {{ANNOTATED_TEXT}} 完全一致",
  "meaning": {
    "word1": "word1 在当前句子中的中文含义",
    "word2": "word2 在当前句子中的中文含义"
  }
}

要求：
- 只能输出 JSON 对象本身，不要输出其他任何文字。
- 必须可被 JSON.parse 直接解析。
- 使用双引号，不要有多余逗号。
